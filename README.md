# 个人记账本

一个简单实用的网页记账本，支持收入和支出管理。

## 功能特点

- ✅ 记录收入和支出
- ✅ 多种分类（工资、奖金、餐饮、交通等）
- ✅ 实时统计总览
- ✅ 分类统计明细
- ✅ 筛选和搜索功能
- ✅ 数据本地存储（浏览器本地存储）
- ✅ 响应式设计，支持手机访问

## 使用方法

直接在浏览器中打开 `index.html` 即可使用。
